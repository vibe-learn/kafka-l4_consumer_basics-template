        # kafka — Консьюмер, группа и оффсеты

        Homework template for the Vibe Learn lesson **l4_consumer_basics** (Консьюмер, группа и оффсеты).

        ## Task

        Реализуй consumer group с manual commit, демонстрирующую at-least-once семантику.
Два инстанса консьюмера читают топик `task_events` (8 партиций). Продьюсер пишет 100
сообщений. Обработка каждого сообщения намеренно медленная (sleep 100 мс). В середине
прогона один инстанс убивается через `kill -9`. После перезапуска CI проверяет:
итоговое число успешно обработанных сообщений >= 100 (переобработка допустима, потеря — нет).

        ## How to work

        1. The Vibe Learn platform creates a copy of this repo in your GitHub account when you click «Начать домашку» on the lesson.
        2. Clone it locally, complete the TODOs in `main.go`, and push.
        3. CI (`.github/workflows/ci.yml`) runs the tests on every push. The Vibe Learn platform listens to the result via webhook and unlocks the lesson when CI passes.

        ## Local prerequisites

        - Go 1.22+
        - Docker + docker-compose (for the local Kafka cluster, when the lesson requires it)

        ## Running

        ```bash
        go test ./...
        ```
